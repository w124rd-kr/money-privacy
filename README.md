## 요약 (한국어)

Juns Money는 **로컬 우선(local-first)** 가계부 앱입니다. 입력한 가계부 데이터는 기기에만 저장되며 개발자 서버로 전송되지 않습니다.

| 항목 | 처리 방식 |
|------|-----------|
| 가계부 데이터(거래·금액·메모·자산·분류) | 기기 내부에만 저장, 전송 안 함 |
| 백업 | 선택 · 사용자 본인 iCloud 또는 파일, AES-GCM 암호화 가능 |
| 사용 통계 | 익명(Aptabase, US). PII·IDFA·쿠키 없음. 설정에서 옵트아웃 |
| 문의·건의 | 사용자가 보낸 메시지 + 선택 이메일 + 앱/OS 버전만 전송(Web3Forms) |
| 알림 | 기기 내부 로컬 생성 |
| 광고/제3자 추적 | 없음, 로그인 불필요 |

## Summary (English)

Juns Money is a **local-first** budget app. Your budget data stays on your device and is never sent to the developer's servers.

| Item | How it's handled |
|------|------------------|
| Budget data (transactions, amounts, memos, accounts, categories) | Stored on-device only, never transmitted |
| Backup | Optional · your own iCloud or files, optional AES-GCM encryption |
| Usage analytics | Anonymous (Aptabase, US). No PII / IDFA / cookies. Opt-out in Settings |
| Feedback | Only your message + optional email + app/OS version (Web3Forms) |
| Notifications | Generated locally on-device |
| Ads / third-party tracking | None; no sign-in required |
